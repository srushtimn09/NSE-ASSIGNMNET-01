# Student Record Management System

## Project Description

This project is a menu-driven Python console application designed to
manage student records. It allows users to add, view, search, and delete
student information efficiently using basic Python concepts.

## Features

-   Add new student records
-   View all student records
-   Search student records by roll number or name
-   Delete student records
-   Data persistence using file handling

## Concepts Used

-   Lists
-   Dictionaries
-   Functions
-   Conditional Statements (if / elif / else)
-   Loops (while, for)
-   Exception Handling (try / except)
-   File Handling
-   String Methods
-   List and Dictionary Methods

## Instructions to Run the Program

1.  Ensure Python is installed on your system.
2.  Download the Python source file (`student_record.py`) and
    `README.md`.
3.  Open a terminal or command prompt.
4.  Navigate to the project folder.
5.  Run the program using: python student_record.py

## Input Guidelines

-   Roll Number: Integer value
-   Name: Alphabetic characters
-   Marks: Integer value

## File Handling Details

-   Student records are stored in a text file.
-   Data is saved automatically when exiting the program.
-   Data is loaded from the file when the program starts.
